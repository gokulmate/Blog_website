# MERN Blog App

A modern Blog Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

* Create Blog
* View All Blogs
* Edit Blog
* Delete Blog
* Responsive UI
* React Router Integration
* REST API Integration
* MongoDB Database

## Tech Stack

### Frontend

* React.js
* React Router DOM
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose

## Installation

### Clone Repository

git clone <repository-url>

### Install Dependencies

Frontend:

cd client

npm install

Backend:

cd server

npm install

### Run Application

Backend:

npm run dev

Frontend:

npm run dev

## API Endpoints

GET /api/blogs

POST /api/blogs

PUT /api/blogs/:id

DELETE /api/blogs/:id
